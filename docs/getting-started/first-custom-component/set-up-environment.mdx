## Consideraciones del sistema operativo

Puedes desarrollar componentes personalizados en cualquier sistema operativo que admita NodeJS (Windows, Linux, MacOS, etc.). Si usas Windows, recomendamos instalar [WSL](https://learn.microsoft.com/en-us/windows/wsl/install).

## Instalar NodeJS

Los componentes personalizados están escritos en NodeJS (con TypeScript superpuesto). Aunque muchas versiones de NodeJS se pueden utilizar para construir componentes personalizados, recomendamos usar la última versión LTS disponible en [NodeJS.org](https://nodejs.org/).

Una vez que hayas instalado NodeJS, asegúrate de poder ejecutar tanto `node --version` como `npm --version`.

## Instalar un entorno de desarrollo integrado (IDE)

Si tienes un editor de código favorito, úsalo. Hemos descubierto que [VS Code](https://code.visualstudio.com/) funciona muy bien para el desarrollo de componentes personalizados, pero algunos de nuestros desarrolladores prefieren [Sublime](https://www.sublimetext.com/) o [neovim](https://neovim.io/).

No es necesario, pero si usas VS Code, encontramos útiles las siguientes extensiones:

- [ES Lint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint) para linting de código
- [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) para el formato de código
- [Version Lens](https://marketplace.visualstudio.com/items?itemName=pflannery.vscode-versionlens) para mostrar la última versión de cada paquete en package.json

VS Code tiene IntelliSense (autocompletado) TypeScript incorporado, pero si quieres utilizar lo último y mejor, también puedes instalar la [extensión TypeScript Nightly build](https://marketplace.visualstudio.com/items?itemName=ms-vscode.vscode-typescript-next).


## Instalar la herramienta de línea de comandos (CLI)

La herramienta de línea de comandos, `prism`, se utiliza para inicializar y publicar componentes personalizados.
Puedes instalarla después de instalar NodeJS y `npm` con el siguiente comando:

```bash title="Install prism"
npm install --global @-io/prism
```

Una vez que hayas instalado `prism`, inicia sesión escribiendo `prism login`.
Se te pedirá que ingreses tus credenciales y luego que autorices a `prism` para acceder a tu cuenta.
Haz clic en **Aceptar**.

Después de iniciar sesión, ejecuta `prism me` para verificar que estás conectado. 

```bash
$ prism me
Name: John Doe
Email: john.doe@example.com
Organization: Example Corp - US Region
Endpoint URL: https://app..io
```
