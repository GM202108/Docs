---
id: use-alert-monitors
title: Utilizar monitores de alerta
description: Los monitores de alerta no son sólo para los problemas, asegúrate de que los utilizas para todo lo necesario.
---

El sistema tiene un monitoreo de alertas incorporado que puedes configurar para cualquier integración. Esto te permite recibir una alerta cuando una instancia enfrenta un problema. O bien, puedes crear un monitor de alertas para que te avise cada vez que se activa una instancia o cuando una instancia se ejecuta con éxito; el monitoreo de alertas no solo funciona para eventos negativos.

También puedes elegir dónde se envían las alertas, ya sea por correo electrónico o cualquier otro medio que estés utilizando para tu sistema de respuesta a incidentes (como [PagerDuty](/monitoring-and-alerting/#sending-incidents-to-pagerduty-with-alert-webhooks) o Slack).
