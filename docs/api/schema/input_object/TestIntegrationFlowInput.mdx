## Input fields

| Argument           | Type                                     | Description                                                                                                     |
| ------------------ | ---------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| `payload`          | [`String`](api/schema/scalar/String.mdx) | The payload to send with the POST request that triggers the Integration Flow Test Instance.                     |
| `contentType`      | [`String`](api/schema/scalar/String.mdx) | The content type of the payload to send with the POST request that triggers the Integration Flow Test Instance. |
| `headers`          | [`String`](api/schema/scalar/String.mdx) | The headers to send with the POST request that triggers the Integration Flow Test Instance.                     |
| `id`               | [`ID`](api/schema/scalar/ID.mdx)         | The ID of the IntegrationFlow to mutate.                                                                        |
| `clientMutationId` | [`String`](api/schema/scalar/String.mdx) | A unique identifier for the client performing the mutation.                                                     |
