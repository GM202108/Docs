## Input fields

| Argument    | Type                                                                                | Description                |
| ----------- | ----------------------------------------------------------------------------------- | -------------------------- |
| `direction` | [`OrderDirection!`](api/schema/enum/OrderDirection.mdx)                             | The direction to order by. |
| `field`     | [`InstanceStepResultOrderField!`](api/schema/enum/InstanceStepResultOrderField.mdx) | The field to order by.     |
