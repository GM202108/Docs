Represents a collection of data that defines a Component Action.

## Input fields

| Argument             | Type                                                                              | Description                                                                    |
| -------------------- | --------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| `key`                | [`String!`](api/schema/scalar/String.mdx)                                         | A string which uniquely identifies the Action in the context of the Component. |
| `display`            | [`ActionDisplayDefinition!`](api/schema/input_object/ActionDisplayDefinition.mdx) | Specifies how the Component Action is displayed.                               |
| `inputs`             | [`[InputFieldDefinition]!`](api/schema/input_object/InputFieldDefinition.mdx)     | The InputFields supported by the Component Action.                             |
| `authorization`      | [`AuthorizationDefinition`](api/schema/input_object/AuthorizationDefinition.mdx)  | Specifies how the Action handles Authorization.                                |
| `terminateExecution` | [`Boolean`](api/schema/scalar/Boolean.mdx)                                        | Specifies whether the Action will terminate Instance execution.                |
| `breakLoop`          | [`Boolean`](api/schema/scalar/Boolean.mdx)                                        | Specifies whether an Action will break out of a loop.                          |
| `allowsBranching`    | [`Boolean`](api/schema/scalar/Boolean.mdx)                                        | Specifies whether the Action will allow Conditional Branching.                 |
| `staticBranchNames`  | [`[String]`](api/schema/scalar/String.mdx)                                        | The static branch names associated with an Action.                             |
| `dynamicBranchInput` | [`String`](api/schema/scalar/String.mdx)                                          | The input associated with dynamic branching.                                   |
| `examplePayload`     | [`JSONString`](api/schema/scalar/JSONString.mdx)                                  | An example of the returned payload of an Action.                               |
