Represents the collection of data that defines an Attachment.

## Input fields

| Argument | Type                                      | Description                              |
| -------- | ----------------------------------------- | ---------------------------------------- |
| `name`   | [`String!`](api/schema/scalar/String.mdx) | The name of the Attachment.              |
| `url`    | [`String!`](api/schema/scalar/String.mdx) | The URL where the Attachment is located. |
