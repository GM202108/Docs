Represents authorization details for a Component.

## Input fields

| Argument   | Type                                        | Description                                                    |
| ---------- | ------------------------------------------- | -------------------------------------------------------------- |
| `required` | [`Boolean!`](api/schema/scalar/Boolean.mdx) | Specifies whether authorization is required for the Component. |
| `methods`  | [`[String]!`](api/schema/scalar/String.mdx) | The list of authorization methods supported by the Component.  |
