Represents a choice for an InputField.

## Input fields

| Argument | Type                                      | Description                          |
| -------- | ----------------------------------------- | ------------------------------------ |
| `label`  | [`String!`](api/schema/scalar/String.mdx) | The label to display for the choice. |
| `value`  | [`String!`](api/schema/scalar/String.mdx) | The value of the choice.             |
