## Input fields

| Argument           | Type                                                                                       | Description                                                                       |
| ------------------ | ------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------- |
| `description`      | [`String`](api/schema/scalar/String.mdx)                                                   | Additional notes about the Instance.                                              |
| `customer`         | [`ID!`](api/schema/scalar/ID.mdx)                                                          | The Customer for which the Instance is deployed.                                  |
| `integration`      | [`ID!`](api/schema/scalar/ID.mdx)                                                          | The Integration that has been deployed for the Instance.                          |
| `name`             | [`String!`](api/schema/scalar/String.mdx)                                                  | The name of the Instance.                                                         |
| `labels`           | [`[String]`](api/schema/scalar/String.mdx)                                                 | The labels that are associated with the object.                                   |
| `configVariables`  | [`[InputInstanceConfigVariable]`](api/schema/input_object/InputInstanceConfigVariable.mdx) | Config variable values that are associated with the Instance.                     |
| `flowConfigs`      | [`[InputInstanceFlowConfig]`](api/schema/input_object/InputInstanceFlowConfig.mdx)         | Configuration data for each IntegrationFlow that is associated with the Instance. |
| `configMode`       | [`String`](api/schema/scalar/String.mdx)                                                   | Desired configuration mode.                                                       |
| `clientMutationId` | [`String`](api/schema/scalar/String.mdx)                                                   | A unique identifier for the client performing the mutation.                       |
