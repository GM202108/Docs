## Input fields

| Argument           | Type                                        | Description                                                     |
| ------------------ | ------------------------------------------- | --------------------------------------------------------------- |
| `grant`            | [`Boolean!`](api/schema/scalar/Boolean.mdx) | Specifies whether to grant or revoke the specified Permission.  |
| `permission`       | [`ID!`](api/schema/scalar/ID.mdx)           | The Permission to grant for the specified object.               |
| `object`           | [`ID!`](api/schema/scalar/ID.mdx)           | The object for which the specified Permission is being granted. |
| `id`               | [`ID`](api/schema/scalar/ID.mdx)            | The ID of the User to mutate.                                   |
| `clientMutationId` | [`String`](api/schema/scalar/String.mdx)    | A unique identifier for the client performing the mutation.     |
