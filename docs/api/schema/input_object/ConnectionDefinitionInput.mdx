Represents a collection of data that defines a Component Connection.

## Input fields

| Argument           | Type                                                                                             | Description                                                                        |
| ------------------ | ------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------- |
| `key`              | [`String!`](api/schema/scalar/String.mdx)                                                        | A string which uniquely identifies the Connection in the context of the Component. |
| `label`            | [`String!`](api/schema/scalar/String.mdx)                                                        | The name of the Connection.                                                        |
| `comments`         | [`String`](api/schema/scalar/String.mdx)                                                         | Additional notes about the Connection.                                             |
| `iconPath`         | [`String`](api/schema/scalar/String.mdx)                                                         | Optional path to icon for this Connection.                                         |
| `oauth2Type`       | [`String`](api/schema/scalar/String.mdx)                                                         | Type of OAuth2 connection, if any.                                                 |
| `oauth2PkceMethod` | [`String`](api/schema/scalar/String.mdx)                                                         | Type of OAuth2 PKCE method, if any.                                                |
| `inputs`           | [`[ConnectionInputFieldDefinition]`](api/schema/input_object/ConnectionInputFieldDefinition.mdx) | Inputs for this Connection.                                                        |
