Allows specifying which field and direction to order by.

## Input fields

| Argument    | Type                                                            | Description                |
| ----------- | --------------------------------------------------------------- | -------------------------- |
| `direction` | [`OrderDirection!`](api/schema/enum/OrderDirection.mdx)         | The direction to order by. |
| `field`     | [`CustomerOrderField!`](api/schema/enum/CustomerOrderField.mdx) | The field to order by.     |
