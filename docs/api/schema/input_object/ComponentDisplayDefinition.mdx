Represents a collection of data that defines how a Component is displayed.

## Input fields

| Argument      | Type                                      | Description                                             |
| ------------- | ----------------------------------------- | ------------------------------------------------------- |
| `label`       | [`String!`](api/schema/scalar/String.mdx) | The name of the Component.                              |
| `description` | [`String!`](api/schema/scalar/String.mdx) | Additional notes about the Component.                   |
| `category`    | [`String`](api/schema/scalar/String.mdx)  | The category of the Component.                          |
| `iconPath`    | [`String`](api/schema/scalar/String.mdx)  | The URL that specifies where the Component icon exists. |
