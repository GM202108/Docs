    Returns the specified ObjectPermissionGrant object.

Access is permitted when any of the following condition(s) are met: 1. Always allowed.

## Input fields

| Argument | Type                              | Description                                 |
| -------- | --------------------------------- | ------------------------------------------- |
| `id`     | [`ID!`](api/schema/scalar/ID.mdx) | The ID of the ObjectPermissionGrant object. |

## Return fields ([ObjectPermissionGrant](api/schema/object/ObjectPermissionGrant.mdx))

###### <b>allowRemove</b> ([Boolean!](api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the ObjectPermissionGrant.

###### <b>allowUpdate</b> ([Boolean!](api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the ObjectPermissionGrant.

###### <b>grantedByRole</b> ([Role](api/schema/object/Role.mdx))

The Role through which the Permission is granted, if applicable.

###### <b>id</b> ([ID!](api/schema/scalar/ID.mdx))

The ID of the object

###### <b>obj</b> ([UUID!](api/schema/scalar/UUID.mdx))

A reference to the object for which the Permission is granted.

###### <b>permission</b> ([Permission!](api/schema/object/Permission.mdx))

The Permission being granted.

###### <b>user</b> ([User!](api/schema/object/User.mdx))

The User for which the Permission is granted.
