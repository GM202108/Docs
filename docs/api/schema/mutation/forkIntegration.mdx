    Forks an Integration.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_manage_integrations]. 2. The signed-in User has any of the following permissions for the access function context object 'customer': [customer_manage_integrations] when a value for 'customer.allow_embedded_designer' is provided in the access function context and equals 'True'.

## Input fields ([ForkIntegrationInput!](api/schema/input_object/ForkIntegrationInput.mdx))

| Argument           | Type                                      | Description                                                 |
| ------------------ | ----------------------------------------- | ----------------------------------------------------------- |
| `name`             | [`String!`](api/schema/scalar/String.mdx) | The name of the Integration.                                |
| `description`      | [`String`](api/schema/scalar/String.mdx)  | Additional notes about the Integration.                     |
| `parent`           | [`ID!`](api/schema/scalar/ID.mdx)         | Parent Integration this Integration was forked from, if any |
| `clientMutationId` | [`String`](api/schema/scalar/String.mdx)  | A unique identifier for the client performing the mutation. |
