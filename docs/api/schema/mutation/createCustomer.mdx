    Creates a new Customer object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_manage_customers, org_crud_customers].

## Input fields ([CreateCustomerInput!](api/schema/input_object/CreateCustomerInput.mdx))

| Argument                | Type                                       | Description                                                                          |
| ----------------------- | ------------------------------------------ | ------------------------------------------------------------------------------------ |
| `name`                  | [`String!`](api/schema/scalar/String.mdx)  | The name of the Customer, which must be unique within the scope of its Organization. |
| `description`           | [`String`](api/schema/scalar/String.mdx)   | Additional notes about the Customer.                                                 |
| `allowEmbeddedDesigner` | [`Boolean`](api/schema/scalar/Boolean.mdx) | Specifies whether this Customer can use the Embedded Designer.                       |
| `labels`                | [`[String]`](api/schema/scalar/String.mdx) | The labels that are associated with the object.                                      |
| `externalId`            | [`String`](api/schema/scalar/String.mdx)   | Allows for mapping an external entity to a record.                                   |
| `clientMutationId`      | [`String`](api/schema/scalar/String.mdx)   | A unique identifier for the client performing the mutation.                          |
