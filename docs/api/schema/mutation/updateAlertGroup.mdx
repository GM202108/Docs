    Updates the specified AlertGroup object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_manage_instances].

| Argument           | Type                                     | Description                                                 |
| ------------------ | ---------------------------------------- | ----------------------------------------------------------- |
| `name`             | [`String`](api/schema/scalar/String.mdx) | The name of the AlertGroup                                  |
| `users`            | [`[ID]`](api/schema/scalar/ID.mdx)       | The users in the AlertGroup.                                |
| `webhooks`         | [`[ID]`](api/schema/scalar/ID.mdx)       | The AlertWebhooks in the AlertGroup                         |
| `id`               | [`ID`](api/schema/scalar/ID.mdx)         | The ID of the AlertGroup to mutate.                         |
