    Removes the specified Customer object.

Access is permitted when any of the following condition(s) are met: 1. The signed-in User has any of the following permissions for the associated Organization: [org_manage_customers, org_crud_customers]. 2. The signed-in User has any of the following permissions for the object: [customer_remove].

## Input fields ([DeleteCustomerInput!](api/schema/input_object/DeleteCustomerInput.mdx))

| Argument           | Type                                     | Description                                                 |
| ------------------ | ---------------------------------------- | ----------------------------------------------------------- |
| `id`               | [`ID`](api/schema/scalar/ID.mdx)         | The ID of the Customer to mutate.                           |
| `clientMutationId` | [`String`](api/schema/scalar/String.mdx) | A unique identifier for the client performing the mutation. |
