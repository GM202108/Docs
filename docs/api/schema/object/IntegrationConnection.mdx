Represents a Relay Connection to a collection of Integration objects.

## Return fields

###### <b>edges</b> ([[IntegrationEdge]!](api/schema/object/IntegrationEdge.mdx))

List of edges containing the nodes in this connection.

###### <b>nodes</b> ([[Integration]!](api/schema/object/Integration.mdx))

List of nodes in this connection.

###### <b>pageInfo</b> ([PageInfo!](api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](api/schema/scalar/Int.mdx))

Total count of nodes available.
