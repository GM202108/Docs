Represents an expression that is used to reference Configuration Variables and results from previous steps.

## Return fields

###### <b>allowRemove</b> ([Boolean!](api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the Expression.

###### <b>allowUpdate</b> ([Boolean!](api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the Expression.

###### <b>id</b> ([ID!](api/schema/scalar/ID.mdx))

The ID of the object

###### <b>meta</b> ([JSONString](api/schema/scalar/JSONString.mdx))

An object that contains arbitrary meta data about an Expression.

###### <b>name</b> ([String!](api/schema/scalar/String.mdx))

The name of the Expression.

###### <b>type</b> ([ExpressionType!](api/schema/enum/ExpressionType.mdx))

The type of the Expression.

###### <b>value</b> ([String!](api/schema/scalar/String.mdx))

The value of the Expression.
