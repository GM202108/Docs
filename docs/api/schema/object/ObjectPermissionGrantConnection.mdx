Represents a Relay Connection to a collection of ObjectPermissionGrant objects.

## Return fields

###### <b>edges</b> ([[ObjectPermissionGrantEdge]!](api/schema/object/ObjectPermissionGrantEdge.mdx))

List of edges containing the nodes in this connection.

###### <b>nodes</b> ([[ObjectPermissionGrant]!](api/schema/object/ObjectPermissionGrant.mdx))

List of nodes in this connection.

###### <b>pageInfo</b> ([PageInfo!](api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](api/schema/scalar/Int.mdx))

Total count of nodes available.
