Represents a Relay Connection to a collection of RequiredConfigVariable objects.

## Return fields

###### <b>edges</b> ([[RequiredConfigVariableEdge]!](api/schema/object/RequiredConfigVariableEdge.mdx))

List of edges containing the nodes in this connection.

###### <b>nodes</b> ([[RequiredConfigVariable]!](api/schema/object/RequiredConfigVariable.mdx))

List of nodes in this connection.

###### <b>pageInfo</b> ([PageInfo!](api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](api/schema/scalar/Int.mdx))

Total count of nodes available.
