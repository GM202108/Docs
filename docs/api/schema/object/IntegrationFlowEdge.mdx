A Relay edge to a related IntegrationFlow object and a cursor for pagination.

## Return fields

###### <b>cursor</b> ([String!](api/schema/scalar/String.mdx))

A cursor for use in pagination.

###### <b>node</b> ([IntegrationFlow](api/schema/object/IntegrationFlow.mdx))

The related object at the end of the edge.
