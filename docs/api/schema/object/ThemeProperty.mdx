Represents a Property of a given type used to style the Organization Theme.

## Return fields

###### <b>allowRemove</b> ([Boolean!](api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can remove the ThemeProperty.

###### <b>allowUpdate</b> ([Boolean!](api/schema/scalar/Boolean.mdx))

Specifies whether the signed-in User can update the ThemeProperty.

###### <b>id</b> ([ID!](api/schema/scalar/ID.mdx))

The ID of the object

###### <b>type</b> ([ThemePropertyType!](api/schema/enum/ThemePropertyType.mdx))

The Type of Theme Property.

###### <b>value</b> ([String!](api/schema/scalar/String.mdx))

The Value of the Theme Property.

###### <b>variant</b> ([ThemePropertyVariant](api/schema/enum/ThemePropertyVariant.mdx))

The Theme Variant this Color will be used with.
