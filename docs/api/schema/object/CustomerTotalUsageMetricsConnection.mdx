Represents a Relay Connection to a collection of CustomerTotalUsageMetrics objects.

## Return fields

###### <b>edges</b> ([[CustomerTotalUsageMetricsEdge]!](api/schema/object/CustomerTotalUsageMetricsEdge.mdx))

List of edges containing the nodes in this connection.

###### <b>nodes</b> ([[CustomerTotalUsageMetrics]!](api/schema/object/CustomerTotalUsageMetrics.mdx))

List of nodes in this connection.

###### <b>pageInfo</b> ([PageInfo!](api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](api/schema/scalar/Int.mdx))

Total count of nodes available.
