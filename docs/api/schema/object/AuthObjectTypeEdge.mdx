A Relay edge to a related AuthObjectType object and a cursor for pagination.

## Return fields

###### <b>cursor</b> ([String!](api/schema/scalar/String.mdx))

A cursor for use in pagination.

###### <b>node</b> ([AuthObjectType](api/schema/object/AuthObjectType.mdx))

The related object at the end of the edge.
