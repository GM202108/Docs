Represents a Relay Connection to a collection of Expression objects.

## Return fields

###### <b>edges</b> ([[ExpressionEdge]!](api/schema/object/ExpressionEdge.mdx))

Contains the nodes in this connection.

###### <b>nodes</b> ([[Expression]!](api/schema/object/Expression.mdx))

List of nodes in this connection.

###### <b>pageInfo</b> ([PageInfo!](api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](api/schema/scalar/Int.mdx))

Total count of nodes available.
