Represents a Relay Connection to a collection of UserLevelConfigVariable objects.

## Return fields

###### <b>edges</b> ([[UserLevelConfigVariableEdge]!](api/schema/object/UserLevelConfigVariableEdge.mdx))

List of edges containing the nodes in this connection.

###### <b>nodes</b> ([[UserLevelConfigVariable]!](api/schema/object/UserLevelConfigVariable.mdx))

List of nodes in this connection.

###### <b>pageInfo</b> ([PageInfo!](api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](api/schema/scalar/Int.mdx))

Total count of nodes available.
