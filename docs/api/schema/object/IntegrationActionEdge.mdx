A Relay edge to a related IntegrationAction object and a cursor for pagination.

## Return fields

###### <b>cursor</b> ([String!](api/schema/scalar/String.mdx))

A cursor for use in pagination.

###### <b>node</b> ([IntegrationAction](api/schema/object/IntegrationAction.mdx))

The related object at the end of the edge.
