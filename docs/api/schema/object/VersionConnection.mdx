Represents a Relay Connection to a collection of Version objects.

## Return fields

###### <b>edges</b> ([[VersionEdge]!](api/schema/object/VersionEdge.mdx))

Contains the nodes in this connection.

###### <b>nodes</b> ([[Version]!](api/schema/object/Version.mdx))

List of nodes in this connection

###### <b>pageInfo</b> ([PageInfo!](api/schema/object/PageInfo.mdx))

Pagination data for this connection.

###### <b>totalCount</b> ([Int!](api/schema/scalar/Int.mdx))

Total count of versions
